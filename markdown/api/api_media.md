```
{"m":"m__4=0>>XCWQW233>https://static.codemao.cn/rose/v0/images/system/demandAlbumLarge.png","mc":"000000","i":"477815689842"}
```

```
{"m":"m__4(媒体卡片类型)>>(卡片作者)>(卡片封面图)","mc":"(卡片颜色)","i":"(随机数)"}

```

```

- 内容解析：
```
&1{"s":"(媒体地址)","d":(媒体时长),"c":"(媒体封面)","n":"(媒体名称)","r":"(媒体作者)","b":"(媒体事件类型)"}
```
